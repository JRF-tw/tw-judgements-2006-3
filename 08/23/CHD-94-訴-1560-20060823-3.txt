、論罪科刑：
  (一)刑法於九十四年二月二日修正公布，並於九十五年七月一日
    開始施行後，有關施行前、後新舊法之適用原則，依最高法
    院九十五年第八次刑事庭會議決議如下：
    新法第二條第一項之規定，係規範行為後法律變更所生新
      舊法律比較適用之準據法，於新法施行後，應適用新法第
      二條第一項之規定，為「從舊從輕」之比較。
    基於罪刑法定原則及法律不溯及既往原則，行為之處罰，
      以行為時之法律有明文規定者為限，必行為時與行為後之
      法律均有處罰之規定，始有新法第二條第一項之適用。
    比較時應就罪刑有關之共犯、未遂犯、想像競合犯、牽連
      犯、連續犯、結合犯，以及累犯加重、自首減輕暨其他法
      定加減原因（如身分加減）與加減例等一切情形，綜其全
      部罪刑之結果而為比較。
    從刑附屬於主刑，除法律有特別規定者外，依主刑所適用
      之法律。
    本案被告己○○、丙○○、子○○行為後，刑法關於連續犯
    、幫助犯、累犯、易科罰金及主刑罰金刑部分之條文，均有
    所修正或刪除，並自九十五年七月一日開始施行。本院依前
    揭最高法院刑事庭會議之決議，就被告三人前述涉及犯行之
    相關論罪科刑條文綜合比較後，認應適用修正前刑法有關連
    續犯、幫助犯、累犯、易科罰金及主刑罰金刑部分之規定（
    新舊法比較情形詳見附表）。至沒收部分，則依主刑所依
    據之法律，適用修正前刑法第三十八條第一項第二款之規定
    ，先予敘明。
  (二)按刑法上之故意，可分為直接故意與不確定故意（間接故意
    ），所謂「不確定故意」，係指行為人對於構成犯罪之事實
    ，預見其發生而其發生並不違背其本意者，刑法第十三條第
    二項定有明文。在現今社會金融帳戶之申請甚為簡易方便，
    亦無信用問題，如有使用帳戶之需要，自可自行開戶，衡情
    當無向他人收購帳戶使用之理。且無正當合理之理由收購與
    其無身分、財產上利害關係者之存摺使用，收購者在無相當
    信任關係下，須冒存摺內款項，遭被收購者以補發存摺、提
    款卡後領用之風險，洵非正當之交易所必要，是以在無相當
    信任關係下，無特定營業場所之人隱匿自己身分而收購不特
    定人之帳戶存摺、提款卡，依一般人通常之知識、經驗，均
    應已認知收購存摺者欲藉收購存摺為財產性犯罪。況邇來利
    用人頭帳戶詐欺取財之犯罪類型層出不窮，該等犯罪，多數
    均利用人頭帳戶作為出入帳戶，並經媒體廣為披載，被告三
    人均係成年且智力成熟之人，對此應知之甚詳。本案被告丙
    ○○、子○○將附表壹所示之帳戶存摺、提款卡販售予被告
    己○○，由被告己○○所屬之帳戶收購集團成員轉賣予不詳
    姓名、年籍之人使用，被告三人均未能供明有何正當之理由
    ，足信其等已預見存摺、提款卡將遭利用為財產上犯罪之工
    具。是以附表壹所示被害人遭詐騙或恐嚇而匯款至被告丙○
    ○、子○○帳戶內等情，應不違被告等人之本意。從而，被
    告等人有容任不詳年籍之人利用其帳戶犯財產上犯罪之不確
    定故意，當無疑義。惟被告三人將帳戶提供他人以傳送簡訊
    或以撥打電話方式實施詐騙，顯係基於幫助他人犯詐欺取財
    之犯意，且所為提供帳戶予他人之行為屬於刑法詐欺取財罪
    構成要件以外之行為，其等既以幫助他人犯罪之意思而參與
    犯罪構成要件以外之行為，應屬幫助犯，而非正犯，起訴意
    旨認被告己○○、丙○○為實施詐欺犯行之共同正犯，尚有
    未洽。又雖被告丙○○所賣出之帳戶中，分別被供作詐欺及
    恐嚇取財兩種犯罪型態使用，惟衡諸目前社會常態，被告應
    僅知悉收購之人係用於詐欺取財之財產犯罪，但尚難認定被
    告二人就該收購帳戶之人會將帳戶分別用於詐欺取財及恐嚇
    取財之行為上有所認識，且實施詐欺之正犯係一人或多人，
    正犯詐欺犯行次數為一次或多次，亦非提供帳戶之被告等人
    所能預見，亦即其等幫助對象之身分是否為共同正犯，及幫
    助對象之犯罪行為是否為連續犯或常業犯，應均非被告三人
    在收購或販賣帳戶時主觀上所能認識之範圍內，依「所知輕
    於所犯，從其所知」之法理，本院就被告三人提供帳戶供他
    人之行為，均論以修正前刑法第三十條第一項、第三百三十
    九條第一項之幫助普通詐欺取財罪。是以起訴書認定被告己
    ○○、丙○○為修正前刑法第三百四十條之常業詐欺罪之正
    犯，及被告子○○為常業詐欺罪之幫助犯，均有未洽，本院
    於社會基本事實同一之範圍內，依法變更起訴法條。至被告
    丙○○多次販售如附表壹所示之帳戶予被告己○○，被告子
    ○○多次將其所申辦如附表壹編號至所示之帳戶存摺交
    由被告丙○○轉賣予被告己○○，被告己○○多次將其所收
    購如附表壹所示之帳戶交給他人轉賣予詐欺犯罪之正犯，被
    告三人多次幫助詐欺取財之犯行，均時間緊接，犯罪構成要
    件相同，顯各係基於概括之犯意反覆為之，俱為連續犯，應
    分別依刑法修正刪除前第五十六條之規定，以幫助詐欺取財
    一罪論處，並各依法加重其刑。起訴書僅記載附表壹編號
    至所示被害人遭詐欺之事實，而未論及附表編號至之
    部分，惟附表壹所示各部分犯罪事實，乃係被告丙○○、子
    ○○幫助正犯所實施之犯罪行為，而各次幫助犯行既有前揭
    連續犯之裁判上一罪關係，基於審判不可分原則，本院自應
    全部審酌。再查：被告己○○、丙○○各有如前揭犯罪事實
    欄所載之前案紀錄，並均已執行完畢等情，有台灣高等法院
    被告全國前案紀錄表在卷可稽，其等於有期徒刑執行完畢五
    年內，再犯本件有期徒刑以上之罪，均為累犯，應各依修正
    前刑法第四十七條之規定遞加重其刑。又被告三人所犯為幫
    助普通詐欺取財罪，已如前述，爰依修正前刑法第三十條第
    二項之規定減輕其刑，且其等均同時有刑罰加重、減輕之事
    由，各應依法先加後減之。
  (三)爰審酌被告三人之平日素行，及渠等提供帳戶非但幫助詐騙
    者遂行詐欺犯行，使詐欺者得以隱匿其真實身分，減少遭查
    獲之風險，更使其肆無忌憚，加深犯罪之猖獗，致使社會經
    濟遭受重大危害，而被告己○○屬收購人頭帳戶集團之一員
    ，其惡性程度最深，被告丙○○除販售之自己之帳戶外，尚
    且為被告子○○代售帳戶，其惡性程度次之，至被告子○○
    僅提供自己所申辦之帳戶予他人使用，其可責性相對較輕，
    另參酌被告己○○坦承大部分犯行，被告丙○○、子○○則
    就全部犯罪事實均坦承不諱之犯後態度等一切情狀，分別量
    處如主文所示之刑，並就被告子○○之部分，依修正前刑法
    第四十一條第一項之規定，諭知易科罰金之折算標準，以示
    儆懲。
  (四)至公意旨雖認被告己○○、丙○○二人另犯洗錢防制法第九
    條第一項之罪嫌，惟按同法第二條規定：「本法所稱洗錢，
    係指下列行為：一、掩飾或隱匿因自己重大犯罪所得財物或
    財產上利益者；二、掩飾、收受、搬運、寄藏、故買或牙保
    他人因重大犯罪所得財物或財產上利益者。」，而所謂重大
    犯罪，係指同法第三條第一項所列十二款之罪名，刑法第三
    百三十九條第一項之詐欺取財罪，並非該法第三條第一項所
    列十二款之罪名之一【按洗錢防制法第三條第一項關於「重
    大犯罪」之定義，於九十五年五月三十日雖有所修正，並自
    同年七月一日起開始施行，但不論依修正前、後之規定，刑
    法第三百三十九條第一項均非該法所定之「重大犯罪」，併
    予敘明】，是以被告三人所為，不得以洗錢罪責相繩，就此
    部分應為無罪之判決，但因起訴意旨認此部分與上開被告己
    ○○、丙○○所為有罪部分之犯行，有裁判上一罪關係，爰
    均不另為無罪之諭知。又起訴書記載被告己○○向被告丙○
    ○、黃姓不詳名字年籍之男子、李姓不詳名字年籍之男子所
    收購之人頭帳戶存摺總計多達二十五至二十七個，而被告丙
    ○○販賣予被告己○○及不詳名字年籍之「王先生」之存摺
    亦多達三、四十個（包括戶名張俊才、黃睿淇、黃健松、康
    家萌、陳秀金等人之金融帳戶存摺、提款卡），另被告子○
    ○亦曾將其向上海銀行台中分行、富邦銀行台中分行、台北
    銀行台中分行、中國信託商業銀行台中分行等多家金融機構
    申辦之存摺交由被告丙○○轉賣予被告己○○，因而認被告
    三人提供人頭帳戶之幫助詐欺犯行不僅附表壹所示之部分。
    惟依幫助犯從屬於正犯之理論，除附表壹所示之帳戶已查知
    遭人使用實施詐欺犯行外，其餘帳戶是否曾供作犯罪使用，
    因欠缺相關證據足以佐證，自無法認定有無正犯之存在，是
    不得以被告等人曾自白提供超過附表壹所示之帳戶，即認定
    其等此部分所為亦該當幫助詐欺之犯行，從而此無法認定之
    部分應為無罪之判決，但此部分縱使成立犯罪，亦與被告等
    人提供附表壹所示帳戶而幫助他人詐欺之部分，有連續犯之
    裁判上一罪關係，爰均不另為無罪之諭知，皆附此敘明。
  (五)關於沒收：
    (1)扣案如附表貳編號、所示之行動電話，均係由「王董
      收購帳戶集團」提供予被告丙○○、己○○作為聯絡賣賣
      帳戶使用之通訊工具，業據其二人於本院審理時陳明在卷
      ，又附表貳編號所示之現金五萬元，則為被告己○○之
      兄李溪泉所交付用以作為收購帳戶之現金，亦據被告己○
      ○於本院審理時供述在卷，上述扣案物均為被告所有，且
      係供犯罪使用之物，均依修正前刑法第三十八條第一項第
      二款之規定宣告沒收【另扣案之行動電話ＳＩＭ晶片卡三
      張，按依國內電信公司一般定型化契約之約定，該晶片卡
      之所有權乃係屬電信公司所有，申請人僅因承租門號而取
      得晶片卡使用權，此晶片卡非被告所有，自不在宣告沒收
      範圍內】。
    (2)至於在被告丙○○身上所查獲之另一支行動電話（摩托羅
      拉C155型），據其稱係自己平日與親友聯絡使用，無法證
      明與本案犯罪有關，爰不予宣告沒收。而在被告丙○○住
      處、己○○身上查扣之帳戶存摺、金融卡等物，均係尚未
      完成收購之帳戶存摺、金融卡，業據被告丙○○、己○○
      於本院及警詢時供述在卷，此部分尚未提供正犯作為詐欺
      犯罪使用，自無法認定與幫助詐欺犯行有關，亦不予宣告
      沒收。

    韋霖陷於錯誤，而分三次匯款至被告丙○○所指定之帳戶內
    （總計匯入三萬元），此後丙○○即失去聯絡，因認被告丙
    ○○此部分之行為，係犯刑法第三百三十九條第一項之詐欺
    取財罪，並認與本案起訴部分，有刑法第五十六條修正刪除
    前所定連續犯之裁判上一罪關係等語。但查此部分併辦之事
    實，與本案起訴事實已相隔逾十個月，且犯罪手法迥異（本
    案被告丙○○係提供人頭帳戶之詐欺罪幫助犯，併案部分為

    告丙○○基於概括犯意下反覆實施之行為，應無連續犯之裁
    判上一罪關係，是以此部分併辦事實非本案起訴效力所及，
    應退回由檢察官另行偵辦處理。
  (二)台灣板橋地方法院目前正在審理之九十五年度訴字第八七○

    起訴書），經本院調取該案相關卷證，查知被告己○○於九

    所以才會拜託我三哥李溪泉讓我加入，從事收購人頭帳戶之

│    │          │解讀，故將舊法之「幫助他人犯罪者，為從犯」之文義，│
│    │          │改為「幫助他人實行犯罪行為者，為幫助犯」。然被告三│
│    │          │人不論依修正前之刑法第三十條第一、二項，抑或依修正│
│    │          │後之刑法第三十條第一、二項，均係幫助犯，且均應減輕│
│    │          │其刑，對被告而言並無有利或不利之情形，不生任何影響│
│    │          │，應依刑法第二條第一項前段，適用修正前之規定。    │
├──┼─────┼─────────────────────────┤
│ 2  │連續犯    │修正前法第五十六條有關連續犯之關定，係將反覆實施之│
│    │          │同類複數犯罪行為論以一罪，並得加重其刑至二分之一。│
│    │          │而修正後已刪除連續犯之規定，在實務上將視各種具體犯│
│    │          │罪類型，對於反覆實施之同類複數犯罪行為，或評價為一│
│    │          │罪，或評價為併罰之數罪（參照該條修法理由）。本案之│
│    │          │多件幫助詐欺犯行，依修正刪除後之規定，應評價為數罪│
│    │          │而分別科刑，顯較被告更為不利，依刑法第二條第一項前│
│    │          │段之規定，自應適用修正前刑法第五十六條之規定。    │
├──┼─────┼─────────────────────────┤
│ 3  │累犯      │刑法第四十七條原規定：「受有期徒刑之執行完畢，或受│
│    │          │無期徒刑或有期徒刑之執行而赦免後，五年以內再犯有期│
│    │          │徒刑以上之罪者，為累犯，加重本刑至二分之一。」，修│
│    │          │正後第四十七條第一項規定：「受徒刑之執行完畢，或一│
│    │          │部之執行而赦免後，五年以內故意再犯有期徒刑以上之罪│
│    │          │者，為累犯，加重本刑至二分之一」。被告己○○、朱書│
│    │          │進於本案所犯係故意為之，其修正前後結果並無不同，依│
│    │          │新法第二條第一項前段規定，仍應適用修正前之規定。  │
├──┼─────┼─────────────────────────┤
│ 4  │易科罰金  │刑法第四十一條第一項前段關於得易科罰金之規定，修正│
│    │          │前係以「因身體、教育、職業、家庭之關係或其他正當事│
│    │          │由，執行顯有困難」為其要件，新法則刪除此要件，修正│
│    │          │理由說明上述規定限制過嚴，欠缺彈性，與易科罰金制度│
│    │          │旨在救濟短期自由刑流弊之旨趣不符，且刪除該要件後使│
│    │          │檢察官得視具體個案審酌宜否易科罰金，更符告易科罰金│
│    │          │制度之意旨。另修正前刑法第四十一條第一項前段配合罰│
│    │          │金罰鍰提高標準條例第二條（現已刪除）之規定，係以「│
│    │          │銀元三百元即新台幣九百元折算一日」為易科罰金之折算│
│    │          │標準，至修正後之刑法第四十一條第一項乃係以「新台幣│
│    │          │一千元、二千元或三千元折算一日」為易科罰金之折算標│
│    │          │準。而觀諸歷年來之實務運作情形，在法院宣告易科罰金│
│    │          │折算標準之案件，檢察官執行時乃以准予易科罰金為原則│
│    │          │，不准易科罰金為例外，是以比較修法前後之規定，應認│
│    │          │修正後刑法第四十一條第一項前段提高易科罰金之折算標│
│    │          │準之適用，對於被告等人而言較為不利，依刑法第二條第│
│    │          │一項前段之規定，應適用修正前之規定。              │
├──┼─────┼─────────────────────────┤
│ 5  │關於刑法分│刑法分則各罪法定本刑中，定有罰金刑部分者，據修正後│
│    │則各罪法定│刑法施行法增訂第一條之一：「中華民國九十四年一月七│
│    │本刑涉及罰│日刑法修正施行後，刑法分則編所定罰金之貨幣單位為新│
│    │金刑者，其│臺幣。九十四年一月七日刑法修正時，刑法分則編未修正│
│    │法定本刑科│之條文定有罰金者，自九十四年一月七日刑法修正施行後│
│    │刑範圍之比│，就其所定數額提高為三十倍。但七十二年六月二十六日│
│    │較適用    │至九十四年一月七日新增或修正之條文，就其所定數額提│
│    │          │高為三倍」及刑法第三十三條第五款將罰金刑修正為：「│
│    │          │新臺幣一千元以上，以百元計算之」等規定，然依被告行│
│    │          │為時之刑罰法律，即罰金罰鍰提高標準條例第一條前段規│
│    │          │定之提高倍數十倍及刑法第三十三條第五款規定之罰金最│
│    │          │低額一元計算，本案被告等人幫助犯刑法第三百三十九條│
│    │          │第一項，而該罪法定本刑有關罰金刑之部分，以修正前之│
│    │          │規定對於被告等人較為有利，依刑法第二條第一項前段，│
│    │          │自應適用修正前之規定。                            │
├──┴─────┴─────────────────────────┤
│總結：經綜合前述修正條文而為比較之結果，應適用修正前刑法之相關規定。│
